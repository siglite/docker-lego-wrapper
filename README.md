# docker-lego-wrapper

[lego](https://github.com/go-acme/lego) using with docker.

## Feature

- Add `lego update` command
- Change default directory to `/etc/letsencrypt`

## Usage

```console
$ git clone https://github.com/siglite/docker-lego-wrapper
$ cd docker-lego-wrapper
$ ./lego
```
